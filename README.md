## üß≠ Project Overview
This project is a **real-world data analytics case study** inspired by **Ola Cabs**, aimed at analyzing ride data to uncover insights into **customer behavior, operational performance, and business efficiency**.  

It integrates **SQL for data analysis** and **Power BI for interactive visualizations**, demonstrating full-cycle data analytics skills ‚Äî from data creation to business insights presentation.

---

## üéØ Objective
- Analyze booking trends, cancellations, and performance metrics.  
- Understand customer and driver satisfaction through ratings.  
- Identify key growth and efficiency improvement opportunities.  
- Present insights via professional Power BI dashboards.

---

## üóÇÔ∏è Dataset Description
The dataset simulates over **100,000 rides in Bengaluru** across one month, with realistic patterns of bookings, cancellations, and ratings.

| **Field** | **Description** |
|------------|-----------------|
| `Date`, `Time` | Booking date and time |
| `Booking_ID` | Unique booking identifier |
| `Booking_Status` | Success / Cancelled / Incomplete |
| `Customer_ID` | Unique customer code |
| `Vehicle_Type` | Auto, Mini, Prime Sedan, Prime SUV, Bike, eBike |
| `Pickup_Location`, `Drop_Location` | 50 dummy locations in Bengaluru |
| `VTAT`, `CTAT` | Vehicle and Customer arrival times |
| `Booking_Value` | Ride fare |
| `Payment_Method` | Cash, UPI, Credit Card, Wallet |
| `Ride_Distance` | Distance (km) |
| `Driver_Ratings`, `Customer_Rating` | Service quality |
| `Cancelled_Rides_by_Customer/Driver` | Cancellation reasons |

**Data Quality Rules:**
- ‚úÖ 62% Successful Bookings  
- ‚ùå <7% Customer Cancellations  
- ‚ùå <18% Driver Cancellations  
- ‚ùå <6% Incomplete Rides  
- üìà Increased weekend bookings and revenue

## üßÆ SQL Analysis
Structured queries were developed to extract insights, create analytical views, and validate data assumptions.

**Sample SQL Views:**
- Retrieve all successful bookings  
- Average ride distance per vehicle type  
- Top 5 customers by booking volume

- Total revenue from successful bookings  
- Maximum and minimum driver ratings by vehicle type

## üìà Key Business Insights
- üöò **Prime Sedan & SUV** generate the **highest average booking values** and distances.  
- üôã‚Äç‚ôÇÔ∏è **Customer cancellations** mainly occur due to **driver delays or requests to cancel**.  
- üí∏ **UPI payments** dominate across successful rides, showing digital preference.  
- üìÖ **Weekend bookings** increase by ~20%, driving higher revenue.  
- üåü **Drivers with higher ratings** show significantly **fewer cancellations and better ride completion rates**.  

---

## üí° Business Recommendations
- Introduce **driver incentives** for timely pickups to reduce cancellations.  
- Promote **UPI and wallet payments** with cashback offers.  
- Focus **marketing campaigns on weekends** and match days.  
- Implement a **customer feedback loop** to improve driver training.  
- Prioritize **vehicle type allocation** (Prime Sedan/SUV) in high-value zones.
